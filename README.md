# Help Desk Ticket System

A full-stack ticket management application built with:

- FastAPI
- Streamlit
- SQLite
- Plotly

## Features

- User Authentication
- Create Tickets
- Update Ticket Status
- Delete Tickets
- Search & Filter Tickets
- Dashboard Metrics
- Priority Analytics
- Status Analytics
- Activity Audit Log

## Tech Stack

Backend:
- FastAPI
- SQLite

Frontend:
- Streamlit
- Plotly

## Skills Demonstrated

- REST API Development
- Database Design
- Authentication
- CRUD Operations
- Data Visualization
- Full-Stack Development
